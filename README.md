# 🎌 AnimeFlux - Explorer

Projeto desenvolvido para a atividade de consumo de API pública, focado em uma experiência **Mobile-First** e design moderno.

## 🔗 Links do Projeto
- **Repositório:** [[Link para este repositório](https://github.com/Everson9/Consumo-de-API)]
- **Deploy:** [[Vercel.com](https://consumoapi-three.vercel.app/)]

## 🛠️ Tecnologias Utilizadas
- **HTML5:** Estrutura semântica.
- **CSS3:** Design responsivo, animações de entrada (Fade-in) e efeitos de Glassmorphism.
- **JavaScript (ES6+):** Manipulação de DOM e consumo assíncrono de API.
- **API:** [Jikan API](https://jikan.moe/) (Dados oficiais do MyAnimeList).
